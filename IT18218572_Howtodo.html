How To Do MIDI Upload


Instructions for the file upload process

1. Do not submit MIDI sequences by others unless you provide a means for contacting the author in order to obtain the necessary permission.
2. Only the following file formats can be uploaded : MIDI files (.mid), Zip archives (.zip-rar).
3. Please use only descriptive file names if possible, with no spaces (use underscore instead of spaces).
Example: malata_bambarekuse.mid.
4. Maximum allowed file size for a upload is 500 KB (KiloBytes).
